class XMLAttributeNotFound : object.Exception;
This exception will be throw when you try to read an attributes which does not exist in current node.

class XMLNodeNotFound : object.Exception;
This exception will be throw when you try to access an XML node which does not exist.

class XMLParseException : object.Exception;
This exception will be throw when parsing XML file or XML literal is faild.
